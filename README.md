# 🧠 Тест Стэнфорд-Бине

## Описание проекта

Программный продукт «Тест Стэнфорд-Бине» представляет собой desktop-приложение для оценки уровня интеллекта детей дошкольного и младшего школьного возраста (от 2 до 14 лет). Приложение реализует адаптивную методику тестирования и позволяет оценить готовность ребенка к обучению в школе.

## Функциональные возможности

### 1. Основные функции
- **Адаптивное тестирование** — сложность вопросов подбирается в зависимости от возраста ребенка
- **60 вопросов** по 7 категориям:
  - Общие знания об окружающем мире
  - Математические способности
  - Развитие речи и грамота
  - Логическое мышление
  - Пространственное мышление
  - Память и внимание
  - Подготовка к школе

### 2. Интерфейс
- Полноэкранный режим
- Пастельная цветовая гамма
- Закругленные кнопки
- Прогресс-бар для отслеживания прохождения теста

### 3. Результаты
- Расчет IQ по формуле: **IQ = 100 + 15 × ((сырой балл − среднее для возраста) / стандартное отклонение)**
- Сохранение результатов в текстовый файл
- История прохождений

### 4. Дополнительные возможности
- Прерывание теста с подтверждением
- Сохранение и просмотр истории результатов
- Детальный отчет по каждому прохождению

## Технические характеристики

### Системные требования
- **Операционная система:** Windows 10/11
- **Процессор:** 1 ГГц или выше
- **Оперативная память:** 512 МБ
- **Свободное место на диске:** 100 МБ
- **Разрешение экрана:** 1024×768 или выше

### Стек технологий
- **Язык программирования:** Python 3.8+
- **Библиотека GUI:** Tkinter (встроенная)
- **Дополнительные библиотеки:** ttk, random, datetime, math, os
